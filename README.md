# Overview

U Devs Donation Collection Platform is a web-based application that connects donors with verified needy individuals and causes. The platform is designed to streamline donation management, track donors and recipients, and provide a clean, user-friendly interface for both donors and administrators.
## Features

Dashboard Overview: Displays total donations, active causes, donors, and verified needy contacts.

Donation Causes: Showcase urgent, verified, and emergency causes with donation progress bars.

Needy Contacts: Manage and view verified recipients with categories (Medical, Education, Emergency).

Donation Givers: Track recent donors, donation type (One-Time / Monthly), and messages.

Reports Section: UI overview of total donations, monthly summaries, and category-wise distribution.

About & Contact: Platform information, developer details, and a contact form (UI only).

Responsive Design: Fully responsive for desktops, tablets, and mobile devices.

Interactive Modals: View cause details and donate (UI preview).

# Usage

Dashboard: See total donations, active causes, donors, and verified needy contacts.

Donation Causes: Explore urgent and verified causes, check progress, and view details.

Needy Contacts: Search and view verified individuals/families who need donations.

Donation Givers: View recent donors, donation types, and messages.

Reports: Preview analytics and donation summaries.

About & Contact: Learn about U Devs and send messages (UI only).

Note: This platform currently functions as a UI prototype. Backend integration for actual donations can be implemented in future versions.

# Technologies Used

Frontend: HTML5, CSS3, Bootstrap 5, JavaScript
